# 🧠 RAG-powered FAQ Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built with Streamlit, FAISS, and Ollama. It allows users to query department-specific FAQs using natural language and get intelligent, context-aware responses.

---

## 🚀 Features

- Semantic search over FAQs using FAISS
- Contextual response generation via Ollama
- Multi-department support (IT, HR, Admin, etc.)
- Chat history saving and loading
- Predefined responses for common greetings
- Index caching and clearing
